‚ö° Installation

System Requirements: Python 3.8+ (Mac, Linux, Windows/WSL, Android via Termux).

The easiest method is using our automated installer script which handles dependencies and path configuration:

curl -sL https://raw.githubusercontent.com/emRival/cli-local-share/main/install.sh | bash

Alternatively, manual installation is supported for advanced users:

git clone https://github.com/emRival/cli-local-share.git
cd cli-local-share
pip3 install .

üöÄ Quick Start

Once installed, the global command sharecli becomes available.

sharecli

This command launches the interactive configuration wizard. No command-line arguments are needed‚Äîeverything is configured via the TUI.

‚öôÔ∏è Setup Wizard

The application guides you through a secure setup process every time you run it:

  • Directory Selection: Choose "Browse" to navigate your filesystem interactively, or type a path manually.
  • Port Configuration: Defaults to 8080. The system automatically detects conflicts and suggests alternatives.
  • HTTPS: Generates a self-signed certificate on the fly for encrypted transport.
  • Session Timeout: Define how long the server remains active (e.g., "10" for 10 minutes). Input 0 for unlimited.
  • Permissions: Explicitly allow or deny Upload and Delete actions for connected clients.

üîê Authentication Modes

We provide three tiers of access control:

Token (Recommended)

Generates a cryptographically secure, random URL token. Users with the link are logged in automatically.

Password

Traditional username/password flow. You define the custom password during setup.

Note: "None" mode is also available but effectively disables security. Use only in trusted testing environments.

üîß Troubleshooting

Port Already in Use?

If port 8080 is busy, the wizard will prevent you from crashing and suggest 8081, 8082, etc.

"Not Secure" Warning?

When using HTTPS with a self-signed certificate, modern browsers will warn you. This is expected. Click "Advanced" -> "Proceed to site" to continue.
