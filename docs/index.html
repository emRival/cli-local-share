‚ö° Installation

The easiest way to install is using our one-line installer:

curl -sL https://raw.githubusercontent.com/emRival/cli-local-share/main/install.sh | bash

Or manually via Git:

git clone https://github.com/emRival/cli-local-share.git
cd cli-local-share
pip3 install .
